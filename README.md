# Cyber Study Quest Ultimate Offline v1.0

這版是「超級強化離線前台版」：

- 不需要 API
- 不需要後台
- 不需要 Netlify
- 不需要 GitHub
- 直接打開 index.html 就能玩
- 五科：國文、英文、數學、自然、社會
- 每科每難度 1000 題：簡單、普通、困難
- 總題庫：15000 題
- 題目為原創題型模板產生，參考會考科目、課綱與常見題型方向，但不複製網路題目
- 自適應難度：連對升難度，連錯降難度
- 錯太多會有懲罰：額外扣 HP、Pixel 能量下降
- 錯題庫、任務報告、貓貓養成
- Nova GIF 角色與對話框

## 使用方式

1. 解壓縮
2. 雙擊 index.html
3. 開始玩

- 連錯仍會降難度：困難 → 普通 → 簡單

- HP 歸零才會強制結束；也可以隨時查看報告

## v1.4：AI 同類型錯題加練

這版新增 AI 輔助出題架構：

- 平常仍可完全離線遊玩
- 答錯後會記錄錯題類型
- 報告頁可以按「Nova，依錯題生成同類型加練題」
- 若已部署 Netlify 並設定 `OPENAI_API_KEY`，會由 AI 生成新題
- 若沒有部署或沒有 API Key，會自動使用離線題庫找同類型題目
- 生成的加練題可以按「把這題加入下一題挑戰」

### Netlify 設定

部署後設定環境變數：

```text
OPENAI_API_KEY = 你的 OpenAI API Key
OPENAI_MODEL = gpt-4.1-mini
```

後台函式位置：

```text
netlify/functions/generate-question.js
```


## v1.5：改用 Google Gemini API

這版已將 AI 同類型錯題加練的後台從 OpenAI 改成 Google Gemini。

後台函式位置：

```text
netlify/functions/generate-question.js
```

Netlify 環境變數請設定：

```text
GEMINI_API_KEY = 你的 Google AI Studio API Key
GEMINI_MODEL = gemini-2.0-flash
```

其中 `GEMINI_MODEL` 可不填，預設會使用：

```text
gemini-2.0-flash
```

如果沒有設定 `GEMINI_API_KEY`，網站仍然可以玩，會自動改用離線同類型題庫。

### Google AI Studio 取得 API Key

到 Google AI Studio 建立 API Key，格式通常類似：

```text
AIzaSy...
```

部署到 Netlify 後，請到：

```text
Site configuration → Environment variables
```

新增 `GEMINI_API_KEY` 後重新 Deploy。
