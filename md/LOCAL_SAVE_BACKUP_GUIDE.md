# v93 本地存檔與備份說明

本版採用「localStorage + JSON 下載備份」雙層存檔。

## 會怎麼存？

按下「存檔並下載備份」後，遊戲會同時做兩件事：

1. 寫入瀏覽器 localStorage
2. 下載一個 JSON 備份檔

## 什麼情況會自動讀到？

只要玩家使用：

- 同一個網站網址
- 同一台裝置
- 同一個瀏覽器
- 沒有清除網站資料

再次打開遊戲時，就會自動讀到 localStorage 裡的進度。

## 什麼情況要用匯入？

如果玩家：

- 換電腦
- 換手機
- 換瀏覽器
- 清除瀏覽器資料
- GitHub Pages 網址變了

就可以按「匯入備份 JSON」，選擇之前下載的 JSON 存檔恢復。

## 注意

GitHub Pages 或 Netlify 不會幫你存玩家資料。這個方案不使用雲端，所以不占用雲端資料庫資源。
